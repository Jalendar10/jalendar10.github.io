# Jalendar Reddy Maligireddy Personal Website

This is a simple GitHub Pages portfolio website.

## How to publish

1. Create a GitHub repository named:

```text
jalendar10.github.io
```

2. Copy your website files into this folder:

```text
index.html
styles.css
script.js
README.md
```

3. Then run:

```text
git add .
git commit -m "Add personal portfolio website"
git push origin main
```
